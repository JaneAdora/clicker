# clicker v2 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Turn clicker into a multi-TV, easy-to-pair terminal remote with live text entry, configurable 0–9 app shortcuts, and swipe-to-D-pad — usable by anyone, not just one setup.

**Architecture:** Extend the existing sync-`main` → tokio-`run()` → connection-task design. Config grows from one flat TV into a device registry (one shared client cert paired with many TVs). New `InputMode` variants (DevicePicker, TextInput) reuse the established mode-dispatch in `app.rs`. A new `discovery.rs` mDNS task and IME message plumbing in `remote.rs` add the network-facing features. Mouse capture in `main.rs` feeds swipe→D-pad mapping in `app.rs`.

**Tech Stack:** Rust, tokio, ratatui 0.29, crossterm 0.28, prost 0.13, serde/toml, `mdns-sd` (new), tokio-rustls (ring).

## Global Constraints

- Rust stable; no system `protoc`, no `cmake` (keep `protox` + `ring`). One line each:
- Every new dependency must be pure-Rust or already transitively present. `mdns-sd` is pure-Rust (chosen for this reason).
- Runtime secrets (cert/key/config) stay in `~/.config/clicker/`; never committed. `.gitignore` already covers `*.pem`/`*.key`/`config.toml`.
- One shared client identity (`cert.pem`/`key.pem`) is paired with every TV. No per-TV cert.
- Single `config.toml`, auto-migrated from the v1 flat shape. clicker owns the file (whole-file writes; comments not preserved).
- Keep all bindings single, un-shifted keys. App shortcuts are digits `1 2 3 4 5 6 7 8 9 0`.
- TDD: every task starts with a failing test where logic is pure; LIVE protocol paths (mDNS browse, IME wire) get unit-tested message/parse construction plus a manual checklist.
- Commit after every task.

```rust
#[derive(Clone, Copy, Debug, Default, PartialEq, Serialize, Deserialize)]
#[serde(rename_all = "lowercase")]
pub enum LaunchKind { #[default] Url, Package }

#[derive(Clone, Debug, PartialEq, Serialize, Deserialize)]
pub struct Shortcut { pub label: String, pub kind: LaunchKind, pub target: String }

#[derive(Clone, Debug, Default, PartialEq, Serialize, Deserialize)]
pub struct DeviceEntry {
    pub id: String,
    pub name: String,
    pub host: String,
    pub paired: bool,
    pub last_volume: Option<u8>,
}

pub fn slugify(name: &str) -> String {
    let mut s = String::new();
    let mut prev_dash = false;
    for c in name.chars() {
        if c.is_ascii_alphanumeric() {
            s.push(c.to_ascii_lowercase());
            prev_dash = false;
        } else if !prev_dash {
            s.push('-');
            prev_dash = true;
        }
    }
    let s = s.trim_matches('-').to_string();
    if s.is_empty() { "tv".into() } else { s }
}
```

Migration strategy: deserialize into an untagged enum `enum AnyConfig { V2(Config), V1(V1Flat) }` where `V1Flat { host: Option<String>, name: Option<String>, paired: bool, last_volume: Option<u8> }`. If V2 parses but has no devices AND the raw text contains a top-level `host =`, treat as V1 (covers the ambiguous empty case). Concretely:

```rust
use std::collections::BTreeMap;

#[derive(Deserialize)]
struct V1Flat {
    host: Option<String>, name: Option<String>,
    #[serde(default)] paired: bool, last_volume: Option<u8>,
}

fn load_from(dir: &std::path::Path) -> Config {
    let Ok(text) = std::fs::read_to_string(dir.join("config.toml")) else {
        return Config::default();
    };
    // v2 first
    if let Ok(cfg) = toml::from_str::<Config>(&text) {
        if !cfg.devices.is_empty() { return cfg; }
    }
    // v1 flat -> migrate
    if let Ok(v1) = toml::from_str::<V1Flat>(&text) {
        if let Some(host) = v1.host {
            let name = v1.name.unwrap_or_else(|| "Android TV".into());
            let id = slugify(&name);
            let mut cfg = Config::default();
            cfg.devices.push(DeviceEntry { id: id.clone(), name, host,
                paired: v1.paired, last_volume: v1.last_volume });
            cfg.last_device = Some(id);
            return cfg;
        }
    }
    Config::default()
}
```

`Config`, helpers, and defaults:

```rust
#[derive(Clone, Default, Serialize, Deserialize)]
pub struct Config {
    #[serde(default)] pub last_device: Option<String>,
    #[serde(default, rename = "device")] pub devices: Vec<DeviceEntry>,
    #[serde(default)] pub shortcuts: BTreeMap<String, Shortcut>,
}

impl Config {
    pub fn active_device(&self) -> Option<&DeviceEntry> {
        let id = self.last_device.as_deref()?;
        self.devices.iter().find(|d| d.id == id)
    }
    pub fn active_device_mut(&mut self) -> Option<&mut DeviceEntry> {
        let id = self.last_device.clone()?;
        self.devices.iter_mut().find(|d| d.id == id)
    }
    pub fn device_by_id_mut(&mut self, id: &str) -> Option<&mut DeviceEntry> {
        self.devices.iter_mut().find(|d| d.id == id)
    }
    pub fn upsert_device(&mut self, d: DeviceEntry) {
        self.last_device = Some(d.id.clone());
        if let Some(slot) = self.devices.iter_mut().find(|x| x.id == d.id) { *slot = d; }
        else { self.devices.push(d); }
    }
    pub fn shortcut(&self, digit: char) -> Option<Shortcut> {
        if let Some(s) = self.shortcuts.get(&digit.to_string()) { return Some(s.clone()); }
        default_shortcuts().get(&digit.to_string()).cloned()
    }
}

pub fn default_shortcuts() -> BTreeMap<String, Shortcut> {
    let url = |label: &str, t: &str| Shortcut { label: label.into(), kind: LaunchKind::Url, target: t.into() };
    let pkg = |label: &str, t: &str| Shortcut { label: label.into(), kind: LaunchKind::Package, target: t.into() };
    BTreeMap::from([
        ("1".into(), url("Netflix", "https://www.netflix.com/title")),
        ("2".into(), url("YouTube", "https://www.youtube.com")),
        ("3".into(), pkg("Disney+", "com.disney.disneyplus")),
        ("4".into(), pkg("Max", "com.wbd.stream")),
        ("5".into(), pkg("Amazon", "com.amazon.amazonvideo.livingroom")),
        ("6".into(), pkg("Hulu", "com.hulu.plus")),
        ("7".into(), pkg("Spotify", "com.spotify.tv.android")),
    ])
}
```
